ML Pipeline Implementation

The complete machine learning pipeline from raw images to final predictions.

1

Data Collection

Download CIFAR-10 dataset from the official source using urllib.

urllib.request.urlretrieve(DATASET_URL, "cifar-10-python.tar.gz")
2

Data Preprocessing

Normalize pixel values, flatten images, and standardize features.

X = X.astype('float32') / 255.0
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)
3

Feature Engineering

Apply PCA to reduce 3072 pixel features to 100 principal components.

pca = PCA(n_components=100)
X_pca = pca.fit_transform(X_scaled)
4

Model Training

Train three models: Logistic Regression, Random Forest, and SVM.

models = {
    "Logistic Regression": LogisticRegression(max_iter=500),
    "Random Forest": RandomForestClassifier(n_estimators=30, max_depth=12),
    "SVM": SVC(kernel='rbf', probability=True)
}
5

Model Evaluation

Evaluate using Accuracy, Precision, Recall, F1-Score, and Confusion Matrix.

accuracy_score(y_test, y_pred)
precision_score(y_test, y_pred, average='weighted')
recall_score(y_test, y_pred, average='weighted')
f1_score(y_test, y_pred, average='weighted')
confusion_matrix(y_test, y_pred)
6

Prediction

Load the saved model and predict the class of a new uploaded image.

img = Image.open(image).resize((32, 32))
features = pca.transform(scaler.transform(img.flatten() / 255.0))
prediction = model.predict(features)
