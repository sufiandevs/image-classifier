About This Project

This project was developed as part of a Machine Learning course. It demonstrates an end-to-end image classification system using traditional ML algorithms.

📚 Project Title

Image Classification System using CIFAR-10 Dataset

🔬 Problem Statement

Classify 32×32 color images into one of 10 categories: airplane, automobile, bird, cat, deer, dog, frog, horse, ship, truck.

💻 Language & Libraries

Language: Python

Libraries: NumPy, Matplotlib, Pandas, scikit-learn, Flask

🧠 Models Used

  • Logistic Regression
  • Random Forest
  • Support Vector Machine (SVM)
