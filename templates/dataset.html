Dataset Used: CIFAR-10

CIFAR-10 is a popular benchmark dataset for image classification. It contains 60,000 tiny 32×32 color images across 10 classes.

🔗 Dataset Source

Original Source: Alex Krizhevsky, University of Toronto

Download Link: https://www.cs.toronto.edu/~kriz/cifar.html

Kaggle Mirror: CIFAR-10 on Kaggle

📁 Dataset Details

  • Total Images: 60,000
  • Training Images: 50,000
  • Test Images: 10,000
  • Image Size: 32 × 32 pixels
  • Color Channels: 3 (RGB)
  • Number of Classes: 10

🐍 Source Code: Loading the Dataset

import os, pickle, urllib.request, tarfile
import numpy as np

DATASET_URL = "https://www.cs.toronto.edu/~kriz/cifar-10-python.tar.gz"
urllib.request.urlretrieve(DATASET_URL, "cifar-10-python.tar.gz")

with tarfile.open("cifar-10-python.tar.gz", 'r:gz') as tar:
    tar.extractall(".")

def unpickle(file):
    with open(file, 'rb') as fo:
        return pickle.load(fo, encoding='bytes')

train_images, train_labels = [], []
for i in range(1, 6):
    batch = unpickle(f'cifar-10-batches-py/data_batch_{i}')
    train_images.append(batch[b'data'])
    train_labels.extend(batch[b'labels'])

train_images = np.vstack(train_images)
train_labels = np.array(train_labels)

test_batch = unpickle('cifar-10-batches-py/test_batch')
test_images = test_batch[b'data']
test_labels = np.array(test_batch[b'labels'])
